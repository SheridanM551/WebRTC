# 如何使用
這是使用WebRTC開發的辨識系統架構，包含一個server負責跑模型，另一個使用者負責上傳畫面以及接收BBOX，使用瀏覽器當作用戶端軟體。
開發過程分成多個階段，可以參考該周的進度簡報的說明。也可以直接使用最後的版本(phase 3)，版本之間互不影響。
